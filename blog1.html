Let‚Äôs take a minute to understand this bolierplate.



First line tells the document type i.e. html here.Second line has an html tag which is the root element i.e. everything is inside it. Html has two children, head and body tags. Inside the head tag is the metadata i.e. information about your website for example the title of your website, the encoding method, etc. and inside body is gonna be all the content(like heading, paragraph, footer, etc.) we wanna display in our website.

I will use the term "tag" from now on.



In the title tag, write <your name> instead of Document. And in the body tag, write the main content that you wanna show on your page. I leave this to your imagination.

More on that in my later blogs. For now we can think of HTML as nothing more than a file where we put all the content.
